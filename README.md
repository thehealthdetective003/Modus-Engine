# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/04e6cef8-b58b-4b07-b579-320ae05d7941

## Run Locally

**Prerequisites:** Node.js on Windows


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Start the app: `npm run dev`

## Timestamped transcription JSON

Phase 2 imports a transcription JSON file generated by your transcription tool. The JSON must contain an exact `duration` and a word-level `words` array. Each word requires `start`, `end`, and either `word` or `text`. The app groups those timestamps into the selected 8- or 10-second scene windows while preserving silent windows and a shorter final scene.
